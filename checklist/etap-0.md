# Etap 0 - Fundamenty (0 - 3 miesiące) - Checklist

Przed rozpoczęciem Etapu 2 upewnij się, że wykonałeś wszystkie poniższe kroki. 

## 1. **TryHackMe**
- [ ] Przejdź przez **Pre Security** (networking, Linux, web basics)
- [ ] Ukończ **Jr Penetration Tester** ścieżkę (30-40 rooms)
- [ ] Wykonaj raporty w stylu OSCP dla ukończonych rooms
- [ ] Podziel się postępami na GitHubie i YouTube

## 2. **Hack The Box**
- [ ] Ukończ **8-10 Easy** maszyn
- [ ] Ukończ **2-3 Medium** maszyn
- [ ] Pracuj na **Linux/Windows** maszynach naprzemiennie
- [ ] Wykonaj raporty w stylu OSCP dla ukończonych maszyn
- [ ] Dokumentuj progres na GitHubie i YouTube

## 3. **VirtualBox**
- [ ] Stwórz **VirtualBox lab** do testów
- [ ] Zainstaluj i przeanalizuj podstawowe narzędzia bezpieczeństwa
- [ ] Napisz własne narzędzia w Pythonie (network scanner, brute-force tool, itp.)
- [ ] Stwórz i przetestuj **Mini AD lab**

## 4. **Dokumentacja**
- [ ] Stwórz foldery na GitHubie:
  - [ ] **Programming**: Kod, skrypty i narzędzia w Pythonie
  - [ ] **THM**: Raporty i postępy z TryHackMe
  - [ ] **HTB**: Raporty i postępy z Hack The Box
  - [ ] **Checklist**: Lista zadań do etapu 0 i postępów

## 5. **Zbiórka materiałów**
- [ ] Zrób backup i zorganizuj materiały w odpowiednich folderach
- [ ] Przygotuj plik `.md` z dokumentacją postępów w repozytorium
- [ ] Upewnij się, że wszystkie raporty i pliki są dostępne publicznie na GitHubie
